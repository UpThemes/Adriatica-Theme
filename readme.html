Getting Started with Aperturious

Aperturious is a flexible, beautiful Tumblr-style WordPress theme with the ability to change the layout/color scheme, post videos, and more!

Features

Aperturious offers some robust but easy-to-use features:

  • Three Beautiful Color Schemes

    In addition to the main dark theme, there are two light themes as well, which can be enabled on the Aperturious Options page.

  • Three Layout Options

    Choose between a left sidebar, right sidebar or no sidebar for a Tumblr-style blog. This can be changed in the theme options page.

  • Simple Video Embedding

    Similar to the Gallery theme, Aperturious offers the ability to share videos using the Simple Video Embedder plugin from Press75.com. Requires installation of Simple Video Embedder plugin, which can be found at Press75.com.

  • Automatic Image Resizing

    Aperturious now uses TimThumb by Darren Hoyt to resize thumbnails and images within posts. That means all you need to do is upload at least a large image for each post and you're ready to go. No more manual image resizing/multiple uploads.

  • Built-In Ad Zones

    A highly requested feature, Aperturious features a number of ad zones that are styled and ready for your ads! Just visit the Aperturious Options page to drop in your ads.

Installing Aperturious

Once you've uploaded the theme to your theme directory in WordPress and activated Aperturious, make sure to set the permissions on the /aperturious/timthumb/cache folder to 777 to allow TimThumb to automatically generate thumbnails for you. Once you've done that, visit the Aperturious Options page to set your color scheme and layout, along with other options for the theme.

Optional Plugins

You should grab the Simple Video Embedder plugin from Press75.com.

Adding Widgets to the Footer or Sidebar

  1. In your WordPress Admin, click Widgets on the sidebar.
  2. From the drop-down box under 'Current Widgets' find 'Primary Aside' or 'Footer 1,2 or 3' and click the little down arrow next to it.
  3. Drag the widgets you'd like to see shown in this area from left to right (Highest Rated, Most Rated, Recent Comments, etc.)
  4. Click 'Save.'
  5. You should now see your widgets in the footer.

Icons used in this theme are from LED24.de and tenbytwenty.com.
